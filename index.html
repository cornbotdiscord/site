About

The sexiest bot to ever exist.

Plus it's made by an epic developer. CornBot is an awesome multi-tool bot available for awesome features in your server, like Moderation, Reporting, and a lot more. CornMusicBot is just a swagger music bot. I mean, who doesn't want to play YouTube music in a Discord VC while chatting with friends? Set the volume to low, then you've fully set the ambience.

Features

MUSIC

CornMusicBot lets you play music in any Discord VC in any server. It's as simple as that! It uses all videos on YouTube, so if you want to hear that brand new Roblox Update #21 video, just run it, and play it in Discord (without the need of a browser open!) Or just invite your friends. (If you have them)

BASIC

CornBot also provides some of the most basic commands, like 8 ball, Rock Paper Scissors, and more.

Start a stopwatch, search the Urban Dictionary, and even more coming soon.

Use !help to see all commands.

FREE

Why pay expensive prices when you can get awesome tools for completely free! Forever! No pay increase, at all. Always free. Should I mention that it is always free?

MODERATION

The most important part of any Discord Server is always Moderation. Our Moderation tools are awesome for any Discord server looking for powerful and easy to use moderation tools, as well as highly descriptive mod logs.
